## 🤹🏻 기술 스택 선정 이유
```sh
- React : 컴포넌트 기반으로 사용한 효율적인 관리
- Node.js : JS로 편안한 프론트 백엔드 동시 개발
- axios : 비동기통신으로 안정적인 API요청
- websocket : 프론트에서 백엔드에 메시지 전달
``````
